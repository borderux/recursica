# Recursica MCP Server - Agent Instructions

If you are an AI Agent operating within this package, you are either **consuming** the MCP server tools to integrate Recursica in a target application, or **contributing** new features/tools to this MCP server.

Please determine your context and read the appropriate guide:

1.  **Consuming the MCP Server**: If you are calling the MCP tools, review `llms.txt` to understand the list of available tools.
2.  **Contributing to the MCP Server**: If you are adding, modifying, or refactoring tools inside this package, read `CONTRIBUTING.md` to see the structure and unit testing instructions.
