# Architecture

## Overview

This document describes the high-level architecture of the `recursica-storybook` application.

## Dependencies

- **`@recursica/storybook-template`**: Used to supply the base configurations and decorators.
- **`@storybook/react-vite`**: The Storybook builder.

## Key Design Decisions

- This app acts as the central visualization and documentation workspace for all Recursica components and tokens.
